# GestionTemperature
Projet personnel de gestion de la temperature


### Commande linux

- Modification du script 
```
nano temp.c
```
- Regénération script 
```
gcc temp.c -o temp
```
- Exécuter le script
```
./temp
```

Si message d'erreur de type : bash: ./tests.sh: Permission non accordée
```
chmod +x temp
```
